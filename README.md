# 🔐 VSecure

VSecure is an AI-powered authentication system using:
- Hand Gesture Recognition ✋
- Face Recognition 👤
- OTP Verification 📩

## 🚀 Features
- Gesture-based login
- Face backup authentication
- Secure vault for files
- Modern UI (Upwork-inspired)

## 🛠 Tech Stack
- React.js
- Node.js
- MongoDB
- MediaPipe
- face-api.js

## 📦 Setup

### Frontend
cd client
npm install
npm start

### Backend
cd server
npm install
npm run dev
